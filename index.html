OpenGL Rendering Engine

I am writing a 3D rendering engine from scratch with C++ and OpenGL. Windowing is done with SDL.
It currently has a Shapes class, Input controller class, Camera class and a Skybox class. I have "integrated" bullet physics(working on a demo to show that!). Currently working on the lighting system.
This project isn't tied to any IDE/build system whatsoever. Building is handled with a simple Makefile. Compilation or building of project can be done by simply running a specified batch file.
This intends to be a full-fledged cross-platform Game Engine with physics, AI and Scene Editor integrated.
Plans are underway to make it support Android and iOS but for now I only have a few demos runnable on Windows platform.
